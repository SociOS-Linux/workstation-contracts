# M2 Runner ↔ Adapter IPC (NDJSON over stdio) — v0.1

Status: Draft  
Date: 2026-04-14  
Transport: JSON-over-stdio (newline-delimited JSON objects; NDJSON)

## Purpose
Provide a stable contract between a **contract-runner** and backend **adapters** (pixi/nix/etc.) implemented as subprocess plugins.

## Framing
- Runner spawns adapter executable: `adapters/<backend>/adapter`
- Runner writes requests (NDJSON) to adapter stdin
- Adapter writes responses (NDJSON) to stdout
- Adapter writes human logs to stderr (non-JSON)

## Envelope (v0.1)
All requests:
- `id` (string) required
- `op` (string) required
- `protocol_version` (string, MAJOR.MINOR) required after hello
- `lane` (object) optional depending on op
- `params` (object) optional

All responses:
- `id` echo
- `op` echo (or "error")
- `ok` boolean
- `result` object|null
- `errors[]` (non-empty if ok=false)
- `warnings[]`

## Handshake
Runner MUST send `hello` immediately after spawning adapter.
Adapter MUST respond before accepting other ops.

Hello request (runner → adapter):
```json
{"id":"h1","op":"hello","protocol_version":"1.0","runner":{"name":"contract-runner","version":"0.1.0"},"required_capabilities":["task_run","lock_hash","lock_validate"],"optional_capabilities":["deps_inventory"],"params":{"strict":false}}
```

Hello response (adapter → runner):
```json
{"id":"h1","op":"hello","ok":true,"protocol_version":"1.0","result":{"adapter":{"name":"pixi","version":"X.Y.Z","license":"MIT"}, "protocol":{"min":"1.0","max":"1.0","supported":["1.0"]}, "capabilities":{"supported":["task_run","lock_validate","lock_hash","env_realize","deps_inventory"],"missing_required":[]}},"errors":[],"warnings":[]}
```

## Core ops (v0.1)
- `info`
- `lock_validate`
- `lock_hash`
- `env_realize`
- `task_run`
- `deps_inventory`

## Error object
```json
{"code":"E_*","message":"human readable","details":{}}
```

## Canonical error codes (starter set)
- Protocol: `E_PROTOCOL_INCOMPATIBLE`, `E_HELLO_REQUIRED`, `E_CAPABILITY_MISSING`
- Lock: `E_LOCK_MISSING`, `E_LOCK_INVALID`, `E_LOCK_HASH_FAILED`
- Env: `E_ENV_REALIZE_FAILED`
- Task: `E_TASK_UNKNOWN`, `E_TASK_FAILED`, `E_TASK_TIMEOUT`, `E_CMD_FAILED`
- FS: `E_PATH_ESCAPE`, `E_WRITE_DENIED`
- Internal: `E_UNSUPPORTED`, `E_INTERNAL`
